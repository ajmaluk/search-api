## API Endpoints Overview

| Endpoint        | What it gives your AI/search                          |
|-----------------|------------------------------------------------------|
| `/multi`        | Multi-engine results + full page content             |
| `/quick`        | Fast results with snippets only (no scraping)        |
| `/scrape?url=`  | Extract clean text from any URL                      |
| `/news`         | Hacker News search (free API)                        |
| `/wiki`         | Wikipedia search + optional full article             |
| `/reddit`       | Reddit posts and discussions                         |
| `/health`       | Check server status                                 |
